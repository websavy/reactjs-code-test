# reactjs-code-test
====================
A reactjs project

Note that this uses;

 - React
 - Webpack
 
 ## Setup
 ```sh
 git clone $THIS_REPO
 cd reactjs-code-test
 npm install
 ```
 
 ## Runing the local development server
 ```sh
 # To run
 npm run build
 # To stop, ctrl + c
 ```
 
 If you have trouble running the server, ensure you're on the latest version of npm.
